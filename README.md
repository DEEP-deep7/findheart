# 像素爱心收集小游戏

## 项目简介

这是一个简单的像素风格网页小游戏，玩家控制像素小人在地图上收集爱心，每次收集爱心时会显示温馨的话语。游戏无需部署，可直接在浏览器中打开游玩，支持PC端和移动端。

## 核心功能

- **像素风格**：采用经典像素艺术风格
- **双端支持**：PC端键盘控制，移动端触摸控制
- **爱心收集**：随机生成爱心，收集后显示温馨话语
- **响应式设计**：适配不同屏幕尺寸
- **可扩展性**：预留了战斗、升级、技能等系统的扩展接口

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript（原生）
- **渲染**：Canvas API
- **字体**：Google Fonts - Press Start 2P（像素风格字体）

## 如何运行

### 方法一：直接打开

1. 下载或克隆本项目到本地
2. 直接双击 `index.html` 文件，在浏览器中打开
3. 开始游戏

### 方法二：本地服务器

1. 下载或克隆本项目到本地
2. 在项目根目录运行本地服务器：
   - Python 3: `python -m http.server 8000`
   - Node.js: `npx http-server -p 8000`
   - PHP: `php -S localhost:8000`
3. 打开浏览器访问：`http://localhost:8000`
4. 开始游戏

## 游戏操作

### PC端
- **方向键**：控制角色上下左右移动
- **WASD键**：同样支持控制角色移动

### 移动端
- **虚拟方向键**：点击屏幕底部的方向键控制角色移动

## 游戏玩法

1. 游戏开始后，地图上会随机生成10个爱心
2. 控制像素小人移动，寻找并收集爱心
3. 每收集一个爱心，屏幕上方会显示一句温馨的话语
4. 收集完所有爱心后，游戏会自动重新生成爱心，继续游玩

## 可扩展性

本项目设计考虑了可扩展性，预留了以下扩展接口：

### 1. 战斗系统
- 在 `js/modules/` 目录下创建 `combat.js`
- 实现敌人AI、战斗逻辑、伤害计算等

### 2. 升级系统
- 在 `js/modules/` 目录下创建 `upgrades.js`
- 实现经验获取、等级提升、属性加点等

### 3. 技能系统
- 在 `js/modules/` 目录下创建 `skills.js`
- 实现技能学习、技能释放、冷却时间等

### 4. 美术资源
- 在 `assets/images/` 目录下添加角色、敌人、场景等图片
- 在 `assets/sounds/` 目录下添加音效和背景音乐

### 5. 配置修改

在 `js/game.js` 文件中可以修改游戏配置：

```javascript
const GAME_CONFIG = {
    canvasWidth: 800,      // 画布宽度
    canvasHeight: 600,     // 画布高度
    playerSize: 32,        // 玩家大小
    heartSize: 16,         // 爱心大小
    playerSpeed: 4,         // 玩家移动速度
    totalHearts: 10,        // 爱心总数
    textDisplayTime: 3000   // 文本显示时间（毫秒）
};
```

## 开发说明

### 代码结构

- **HTML**：游戏布局和UI结构
- **CSS**：样式设计和响应式布局
- **JavaScript**：
  - 游戏核心逻辑
  - 玩家控制
  - 碰撞检测
  - 渲染系统
  - 模块化扩展接口

### 开发建议

1. **美术资源**：建议使用像素风格的图片，保持游戏整体风格统一
2. **代码规范**：遵循模块化设计，保持代码清晰可维护
3. **性能考虑**：注意Canvas绘制性能，避免不必要的计算
4. **测试**：在不同设备和浏览器上测试游戏兼容性
